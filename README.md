# Gin Rummy (source code)
Ben Brackenbury & Sonny Ash

## Build & Run
1. Clone the project\
  `git clone https://github.com/benbrackenbury/Gin-Rummy.git`

2. Install required dependencies with\
  `npm i` or `yarn`
  
3. Run with\
  `npm start` or `yarn start`
  
* Build with\
  `npm build` or `yarn build`
